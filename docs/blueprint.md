# **App Name**: LoveFilm

## Core Features:

- Video Upload: Allow the user to upload the video they prepared.
- Audio Uploads: Enable the user to upload two different audio narration options for the video.
- Video Playback with Audio Selection: Implement a video player that allows the user to play the video while selecting one of the two audio narration options.  Audio will play at the same time as video.
- Simple File Storage: Provide secure cloud file storage for the video and audio narrations.

## Style Guidelines:

- Primary color: Soft rose (#F4B4C3) to evoke romance and tenderness.
- Background color: Very light, desaturated pink (#F8E8ED).
- Accent color: Light purple (#D0B4F4) for a touch of elegance.
- Font: 'Alegreya', a serif font with an elegant and contemporary feel.
- Use simple, heart-themed icons to indicate controls such as play, pause, and audio selection.
- Clean, full-screen layout with video in a prominent display area.  Place audio options clearly below.
- Gentle transitions when switching between audio narrations.
